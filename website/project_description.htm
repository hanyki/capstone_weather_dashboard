Our project is to create a web service where Auto-Owners Insurance Agents can enter a location (zip code, address, etc.) and see a list of recent catastrophic weather events. The list will include hail, high winds, snowfall, large amounts of rain, etc.

The list of weather events will come from various weather services include NOAA, wunderground.com, and weatherbug.com. The user will also be able to preserve the data returned from the weather sites.

Users will be able to sort the list of events by type of event, and filter by how long ago the weather even occured.
